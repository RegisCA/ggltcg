## Prerequisites

- Node.js 18+ and npm
- Backend server running on `http://localhost:8000`

## Setup

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Start development server**:
   ```bash
   npm run dev
   ```

3. **Open browser**:
   Navigate to `http://localhost:5173`

## How to Play

1. **Select Deck**: Choose 6 unique cards from the 18-card pool
2. **Play Cards**: Click cards in your hand to see play options
3. **Tussle**: Attack opponent's toys or make direct attacks
4. **AI Opponent**: AI automatically takes its turn
5. **Win**: Sleep all opponent's cards!
